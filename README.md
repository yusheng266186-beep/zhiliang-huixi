# 质量慧析

面向荣县一中高2024级的本地考试质量分析系统。导入领导制作的质量复盘 Excel 后，可按考试、物理类/历史类、班级、学科和学生下钻分析。

## 启动

要求 Node.js `>=22.13.0`。

```bash
npm install
npm run dev
```

打开 <http://localhost:3000>。

## 主要功能

- 年级驾驶舱、年级总成绩、班级横向对比
- 一本/本科上线人数、上线率、学科有效人数与有效率
- 一本/本科临界生分别统计和薄弱学科诊断
- 学生画像、历次考试、小题得分率与知识点分析
- Word、PDF报告直接下载
- 橙色一本、绿色本科的统一视觉语义，以及非线性过渡动画
- 数据健康度、自动去重、缺失字段降级和损坏文件保护
- 数据只保存在本机浏览器 IndexedDB 中

## 本届固定口径

- 7班源表“英语”成绩按“日语”展示和分析。
- 9班源表“生物”成绩及小题按“地理”展示和分析。
- 物理类、历史类混合查看时，每名学生按自己的类别分数线判断上线，不使用平均分数线。
